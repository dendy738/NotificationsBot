Simple Teleram Bot for remaind you about your event via messages.
You can choose type of event, enter the date and time when your event is started and bot will send message to you before 1 hour, 30 minutes, 15 minutes and exactly at moment which you sent to bot.

Also, you can set several events at several dates and times and bot will remind you of all your events.

Using:
  - command:
 

      pip install -r requirements.txt


  - Search bot in the Telegram by tag @your_private_helper_bot.
  - Run code in your IDE.
  - Enjoy it!
